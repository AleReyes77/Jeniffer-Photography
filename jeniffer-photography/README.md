# Jeniffer Photography Website

Sitio web moderno con efectos visuales avanzados para el estudio de fotografía de Jeniffer.

## Características

- Diseño holográfico y efectos 3D
- Carrusel cuántico interactivo
- Sistema de partículas en tiempo real
- Modo claro/oscuro
- Totalmente responsive
- Optimizado para SEO
- Efectos de sonido visuales
- Experiencia de realidad aumentada simulada
